Overview

Alternative functions for getting current and historical stock quotes from Google Finance.


Public Variables and Functions


get-historical-quotes

function
Usage: (get-historical-quotes stock-symbol start-date end-date)

Returns a sequence of historical stock quotes corresponding to the supplied stock symbol, one quote per day between the supplied start and end dates (as org.joda.time.LocalDate objects). Quotes corresponding to dates falling on weekends and holidays are not included in the resulting sequence. If quotes for the given symbol or period cannot be found, it returns nil. The supplied stock symbol can have an optional exchange prefix (.e.g "GOOG" or "NASDAQ:GOOG").

NOTE: The Google Finance service limits the result to 4,000 items. If the supplied date range results in more than that limit, only the most recent 4,000 quotes will be returned.

get-quote

function
Usage: (get-quote stock-symbol)

Returns the current stock quote corresponding to the supplied stock symbol. The result is a map with :symbol, :exchange, :name, :currency, :previous-close, :open, :low, :high, :last (trade), :last-date-time, :change, :percent-change, :volume, and :avg-volume keys. If data for the supplied stock symbol cannot be found, it returns nil.

get-quotes

function
Usage: (get-quotes & stock-symbols)

Returns a sequence of stock quotes corresponding to the given stock symbols. A stock quote is a map with :symbol, :exchange, :name, :currency, :previous-close, :open, :low, :high, :last (trade), :last-date-time, :change, :percent-change, :volume, and :avg-volume keys. If data for a symbol cannot be found, its place in the result sequence will be nil.

parse-historical-quotes

function
Usage: (parse-historical-quotes s)

Parses a string of CSV-encoded historical stock quotes and returns them as a sequence of HistoricalQuote records. It expects one quote per line with fields for date, open, high, low, close, and volume. The first line is assumed to be column headers and is discarded.
